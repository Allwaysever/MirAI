Syarat & Ketentuan

Tanggal Efektif: 8 Oktober 2025

Harap baca Syarat dan Ketentuan ("Ketentuan", "Syarat & Ketentuan") ini dengan saksama sebelum menggunakan aplikasi web MirAI ("Layanan") yang dioperasikan oleh Allwaysever ("kami", "kita", atau "milik kami").

Akses Anda ke dan penggunaan Layanan ini dikondisikan pada penerimaan dan kepatuhan Anda terhadap Ketentuan ini. Ketentuan ini berlaku untuk semua pengunjung, pengguna, dan pihak lain yang mengakses atau menggunakan Layanan. Dengan mengakses atau menggunakan Layanan, Anda setuju untuk terikat oleh Ketentuan ini. Jika Anda tidak setuju dengan bagian mana pun dari ketentuan ini, maka Anda tidak boleh mengakses Layanan.

1. Definisi

  • Layanan: Merujuk pada aplikasi web MirAI yang dapat diakses melalui peramban (browser).
  • Pengembang: Merujuk pada Allwaysever, pencipta dan pengelola Layanan.
  • Pengguna: Merujuk pada Anda, individu atau entitas yang mengakses atau menggunakan Layanan.
  • Layanan Pihak Ketiga: Merujuk pada layanan atau konten (termasuk data, informasi, produk, atau layanan) yang disediakan oleh pihak ketiga yang mungkin ditampilkan atau diakses melalui Layanan, secara spesifik Google Gemini API.
  • Konten Pengguna: Merujuk pada semua data, teks, atau materi lain yang Anda kirimkan sebagai input ke dalam Layanan.
  • Konten Buatan AI: Merujuk pada teks, data, atau respons lain yang dihasilkan oleh Layanan sebagai balasan atas Konten Pengguna.

2. Deskripsi Layanan

MirAI adalah antarmuka eksperimental yang memungkinkan Pengguna untuk berinteraksi dengan model bahasa besar yang disediakan oleh Google (Gemini). Fungsi utamanya adalah untuk memberikan respons berbasis teks berdasarkan input dari Pengguna. Pengguna mengakui bahwa Layanan bersifat probabilistik dan respons yang dihasilkan (Konten Buatan AI) mungkin tidak selalu akurat, lengkap, atau sesuai. Pengguna disarankan untuk melakukan verifikasi mandiri atas semua informasi yang diterima dari Layanan.

3. Kewajiban dan Perilaku Pengguna

Sebagai syarat penggunaan, Anda setuju untuk tidak menggunakan Layanan untuk tujuan apa pun yang melanggar hukum atau dilarang oleh Ketentuan ini. Anda setuju untuk mematuhi semua hukum dan peraturan lokal, negara bagian, nasional, dan internasional yang berlaku. Anda juga setuju untuk tidak:

  1. Mengirimkan atau memasukkan Konten Pengguna yang bersifat ilegal, mengancam, melecehkan, memfitnah, cabul, atau melanggar hak kekayaan intelektual pihak lain.
  2. Mencoba merekayasa balik (reverse-engineer), mendekompilasi, membongkar, atau mencoba menemukan kode sumber dari Layanan.
  3. Menggunakan sistem otomatis, termasuk "robot", "spider", atau "scraper" untuk mengakses Layanan untuk tujuan apa pun tanpa izin tertulis dari kami.
  4. Secara sengaja membebani infrastruktur Layanan secara tidak wajar.
  5. Memberikan informasi pribadi yang sensitif (misalnya, nomor KTP, informasi keuangan, data kesehatan) ke dalam Layanan.

4. Penggunaan Kunci API (API Key)

Layanan ini memerlukan kunci API untuk berkomunikasi dengan Layanan Pihak Ketiga (Google Gemini API).

  • Kunci API Default: Layanan menyediakan kunci API default untuk kemudahan penggunaan. Kunci ini mungkin memiliki batasan kuota atau penggunaan yang dapat berubah tanpa pemberitahuan. Pengembang tidak menjamin ketersediaan atau kinerja kunci API default.
  • Kunci API Pribadi: Pengguna memiliki opsi untuk menyediakan kunci API mereka sendiri. Jika Anda memilih untuk menggunakan kunci API pribadi, Anda sepenuhnya bertanggung jawab atas semua aktivitas yang terkait dengannya, termasuk kepatuhan terhadap syarat dan ketentuan Google, serta potensi biaya apa pun yang mungkin timbul dari penggunaannya.

5. Privasi dan Penanganan Data

Kami menghargai privasi Anda. Kebijakan kami adalah sebagai berikut:

  • Penyimpanan Data Lokal: Layanan ini menggunakan localStorage pada peramban web Anda untuk menyimpan data seperti riwayat percakapan, preferensi tema, bahasa, persona, dan kunci API pribadi Anda. Data ini hanya ada di perangkat Anda dan tidak pernah dikirim atau disimpan di server milik Pengembang.
  • Transmisi Data ke Pihak Ketiga: Ketika Anda mengirimkan prompt (Konten Pengguna), konten tersebut dan riwayat percakapan yang relevan akan dikirim ke Google untuk diproses oleh Gemini API. Penggunaan data ini diatur oleh Kebijakan Privasi Google. Pengembang tidak memiliki akses atau kontrol atas bagaimana Google memproses data ini.
  • Tidak Ada Pengumpulan Data oleh Pengembang: Kami tidak mengumpulkan, menyimpan, atau menganalisis percakapan atau data pribadi Anda.

6. Konten Buatan AI

Kepemilikan dan hak atas Konten Buatan AI tunduk pada ketentuan dari Layanan Pihak Ketiga (Google). Pengembang tidak mengklaim kepemilikan apa pun atas Konten Buatan AI yang Anda hasilkan. Anda bertanggung jawab penuh untuk memastikan bahwa penggunaan Anda atas Konten Buatan AI mematuhi hukum yang berlaku dan tidak melanggar hak pihak ketiga mana pun.
