Frequently Asked Questions (FAQ)

What is Vanilla Stack?

Vanilla Stack means MirAI is built without using heavy frameworks or libraries like React, Vue, or Angular. We only use pure (vanilla) HTML, CSS, and JavaScript, plus Tailwind CSS for utility styling, making it very lightweight.

Is MirAI free?

MirAI is an open-source interface that you can use for free. However, MirAI uses the Google Gemini API which may require a paid API key from Google for advanced usage (depending on current Google Cloud policies).

How does SuperRoundedUI (SRUI) work?

SRUI is a design principle focused on using a consistent 25px border-radius, smooth blur effects (backdrop-filter) on backgrounds, and dark colors that are comfortable for the eyes. This provides a modern, premium, and cohesive look. SRUI is an exclusive UI design system from Allwayseverâ„¢.

What are the Slash Commands?

Slash commands are shortcuts like `/code` or `/tools` that let you quickly set a specific mode or persona for MirAI. This helps the AI focus its response for maximum accuracy and speed on specialized tasks.

Do I need my own API Key?

For the best performance and security, it is recommended to use your own Google Gemini API Key. You can enter this key directly in the Settings menu of MirAI.
